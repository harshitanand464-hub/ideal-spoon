Contact Book Management System (Python & MySQL)
This project is a simple CRUD-based Contact Book application developed using Python and MySQL. It allows users to add, view, update, and delete contacts through a menu-driven console interface and demonstrates database connectivity and SQL query handling.
Features
•	Add new contacts
•	View all contacts
•	Update existing contacts
•	Delete contacts
•	MySQL database integration
•	Menu-driven console application
•	Basic error handling
Technologies Used
•	Python
•	MySQL
•	mysql-connector-python
Database Setup
1. Open MySQL Server or MySQL Workbench.
2. Create a database named 'contact_book'.
3. Run the SQL script provided in the database.sql file to create the table.
How to Run the Project
1. Install required dependency using:
   pip install mysql-connector-python

2. Update MySQL credentials in contact_book.py.

3. Run the application using:
   python contact_book.py
